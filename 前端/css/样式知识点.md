1. 文字上下居中

   ~~~css
   .item{
       height: 25px;
       line-height: 25px;
   }
   要让一个文字在父元素中垂直居中，只需将父元素的line-height设置为一个和父元素height一样的值
   ~~~

2. 鼠标移动上去显示样式

   ~~~css
   .item:hover{
   	background-color: #d9d9d9;
   }
   ~~~

3. a标签去除下划线

   ~~~css
   a{
       text-decoration: none;
   }
   ~~~

4. 字体

   ~~~css
   .news-title a{
       /*加粗*/
       font-weight: bold;
       color: #40406B;
   }
   ~~~
